# Visualizations

This folder contains plots and charts generated during the exploratory data analysis of historical cherry blossom (sakura) bloom dates in Japan.

## Included Visualizations
- Line charts showing first bloom dates over time
- Line charts showing full bloom dates over time
- Comparative plots between first and full bloom timing
- Trend visualizations highlighting long-term shifts

Each visualization is saved as an image file (PNG) and referenced in the main project README where appropriate.
